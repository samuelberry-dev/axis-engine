---

## Quick Start
```bash
git clone https://github.com/samuelberry-dev/axis-engine
cd axis-engine

# Windows
.\gradlew.bat run
# macOS / Linux
./gradlew run
```

> You should see a window with a rotating textured cube.

## Requirements
- Java 17+
- Gradle 8.9+ (wrapper included)
- LWJGL 3.3.4 (resolved via Gradle)
